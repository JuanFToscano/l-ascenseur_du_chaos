# TP01 - Aventure Textile - L'Ascenseur du Chaos

Dans cette histoire, vous incarnez une personne venue séjourner à l’hôtel Malice et Délice après un long voyage. Votre chambre se trouve au quatrième étage, et pour y arriver, vous devez prendre l’ascenseur. Mais cet ascenseur n’est pas comme les autres : il peut vous mener à des situations de vie ou de mort, en vous donnant de précieuses leçons de vie. Bonne chance !


## Comment tester

- Ouvrir `index.html` dans un navigateur **OU**
- Voir la démo : https://juan-f-toscano-itch.io/l-ascenseur-du-chaos
