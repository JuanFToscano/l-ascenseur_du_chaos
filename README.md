# l-ascenseur_du_chaos
Dans cette histoire, vous incarnez une personne venue séjourner à l’hôtel Malice et Délice après un long voyage. Votre chambre se trouve au quatrième étage, et pour y arriver, vous devez prendre l’ascenseur. Mais cet ascenseur n’est pas comme les autres : il peut vous mener à des situations dangereuses. Bonne chance !
